# PRISMA 2020 Checklist

Checklist status: draft, pending final manuscript page/section locations.

This file is a filled working checklist for the review package. It paraphrases
PRISMA 2020 reporting topics and maps them to the current manuscript/repository
locations. The official PRISMA 2020 checklist is available from the PRISMA
statement website: https://www.prisma-statement.org/prisma-2020-checklist

| Item | Reporting topic | Location |
|---|---|---|
| Title | Identify the report as a systematic review | Title |
| Abstract | Structured summary of objectives, methods, results, and limitations | Abstract |
| Rationale | Explain why the review is needed | Section 1 |
| Objectives | State review questions | Section 3.1; `protocol.md` |
| Eligibility criteria | Specify inclusion and exclusion criteria | Section 3.3; `protocol.md` |
| Information sources | List databases, registers, and dates searched | Section 3.2; `search/query_log.csv` |
| Search strategy | Provide complete search strings | Appendix A; `search/query_log.csv` |
| Selection process | Explain screening process and reviewer roles | Section 3.4; `screening/screening_sheet.csv` |
| Data collection process | Explain extraction process and reviewer roles | Section 3.6; `extraction/extraction_matrix.csv` |
| Data items | Define all extracted fields | Section 3.6; `extraction/extraction_matrix.csv` |
| Study quality | Define quality/risk-of-bias assessment | Section 3.5; `quality/qa_rubric.md` |
| Effect measures | Explain why meta-analysis effect measures are not applicable | Section 3.7 |
| Synthesis methods | Describe narrative synthesis procedure | Sections 3.7 and 5 |
| Reporting bias | Discuss reporting bias and missing evidence | Section 8 |
| Certainty | Explain how QA scores affect confidence | Section 3.5 |
| Study selection results | Report flow counts and exclusions | Figure 1; `extraction/search_log.csv` |
| Study characteristics | Present per-study characteristics | Appendix B; `extraction/extraction_matrix.csv` |
| Quality results | Present quality scores per study | Appendix B |
| Individual study results | Present extracted evidence per study | Appendix B |
| Synthesis results | Present cluster and RQ-level findings | Sections 4-6 |
| Reporting bias results | Present reporting-bias observations | Section 8 |
| Certainty results | Present confidence limits from QA | Section 8 |
| Discussion | Interpret findings, limitations, and implications | Section 8 |
| Registration/protocol | Provide registration or protocol location | `protocol.md`; OSF/anonymous mirror pending |
| Support | Describe funding or support | Manuscript funding statement pending |
| Competing interests | Describe competing interests | Manuscript declaration pending |
| Availability | State where data, code, and materials are available | Repository and anonymized mirror pending |
