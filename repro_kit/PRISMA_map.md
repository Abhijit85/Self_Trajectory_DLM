## Pre-registration note
PRISMA expects the protocol to be fixed *before* the search. For a CS venue you can:
- deposit a dated protocol document on **OSF** (osf.io) and cite the (anonymised) link, or
- timestamp the protocol as an arXiv preprint / a dated commit in the (anonymous) repo.
Doing this before you run the queries is what lets you honestly claim the RQs and
criteria were pre-registered.
