# AI Cross-Check Outputs

These files provide a complete review workflow scenario.

- `search/query_log.csv`: raw search count scenario.
- `screening/screening_sheet.csv`: screening sheet scenario.
- `quality/qa_rater1.csv` and `quality/qa_rater2.csv`: profile-based QA scores.
- `quality/reviewer_a_mechanism_reviews.csv`: mechanism-focused review rationales.
- `quality/reviewer_b_evidence_reviews.csv`: evidence-focused review rationales.
- `quality/reviewer_profiles.md`: profile definitions and example judgments.
- `extraction/extraction_matrix.csv`: consensus extraction scenario.
- `extraction/search_log.csv`: PRISMA-style funnel scenario.
